# IITG Summer Analytics 2026 — E-commerce Lead Classification

Binary classification on an e-commerce dataset to identify high-value leads.

## Pipeline
- EDA & data exploration
- One-hot encoding + median imputation
- Feature engineering: page_per_time, products_per_page, Age_Group, Hot_Lead
- Models: Logistic Regression, Random Forest, XGBoost
- Best F1: ~0.54 (Logistic Regression)

## Stack
Python · Pandas · NumPy · scikit-learn · XGBoost · Matplotlib

## Files
- `report.pdf` — exploratory analysis
- `notebook.ipynb` — full ML pipeline
- `submission.csv` — final predictions
